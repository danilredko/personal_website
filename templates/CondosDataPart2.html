Now, let's deal with sqft variable. Instead of having a range we will change it to an average. We will use a simple function.



def sqft_average(sqft):

    if sqft=='0':

        return 0.0

    elif '-' in sqft:

        first_number, second_number = re.findall('\d+', sqft)
        return (int(first_number)+int(second_number))/2

    else:

        return float(sqft)


Now, let's apply this function to sqft.



df['sqft'] = df['sqft'].map(sqft_average)

Let's make a function to add a new variable floor we should have an effect on price.



def get_floor(unit):

    if len(unit)==4:

        return float(unit[0:2])

    elif len(unit)==3 or len(unit)==1:

        return float(unit[0])


Let's consider the variable only with integers values of unit.
